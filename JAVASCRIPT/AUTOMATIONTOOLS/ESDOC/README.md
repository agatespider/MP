## 개요
Javascript 주석을 통한 document 생성
    
## 사이트주소

  1. site : https://esdoc.org/
  
## npm 설치 및 간략 실행
    
    # install ESDoc from npm
    npm install -g esdoc
    
    # move to a your project directory
    cd your-project/
    
    # write a configuration file.
    echo '{"source": "./src", "destination": "./doc"}' > .esdoc.json
    
    # run ESDoc
    esdoc
    
    # see a documentation
    open ./doc/index.html
